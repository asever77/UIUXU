Accordion

Accordion 컴포넌트는 접을 수 있는 콘텐츠 패널을 표시하는 접근성 좋고 유연한 방법을 제공합니다.
이를 통해 사용자는 콘텐츠 섹션을 표시하거나 숨길 수 있어 FAQ, 제품 상세 정보 또는 정보가 효율적으로 구성되어야 하는 모든 시나리오에 이상적입니다.

 HTML 

<div data-accordion="name">
	<div data-accordion-item="name">
		<div data-accordion-head="name">
			<h3 data-accordion-title="name">title 1</h3>
			<button type="button" data-accordion-button="name"></button>
		</div>
		<div data-accordion-body="name" hidden>
			<div data-accordion-section="name">
				...
			</div>
		</div>
	</div>
</div>
Options

Tab 클래스 생성자에 전달할 수 있는 옵션에 대한 자세한 설명입니다.

옵션명 타입 기본값 설명
id String 필수 아코디언 인스턴스의 고유 식별자입니다. 이는 HTML의 주 아코디언 컨테이너 요소에 있는 data-accordion 속성과 일치해야 합니다.
expanded string 또는 null null 컴포넌트 로드 시 어떤 아코디언 항목이 초기에 확장되어야 하는지 지정합니다. 확장하려는 아코디언 버튼의 ID를 제공해야 합니다. null로 설정하면 기본적으로 어떤 항목도 확장되지 않습니다.
singleOpen boolean true 한 번에 하나의 아코디언 항목만 열릴 수 있는지(true) 또는 여러 항목이 동시에 열릴 수 있는지(false)를 결정합니다. true인 경우, 한 항목을 열면 현재 열려 있는 다른 항목이 자동으로 닫힙니다.
scrollIntoView boolean false 스크롤이벤트
scrollOptions object false { behavior: 'smooth', block: 'start' }
