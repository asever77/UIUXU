HTML 구조

아코디언을 사용하려면 특정 data-accordion 속성을 사용하여 HTML을 설정해야 합니다. 각 아코디언 인스턴스는 고유한 data-accordion ID를 필요로 합니다. 내부적으로 각 접을 수 있는 항목은 부모 아코디언의 ID와 일치하는 data-accordion-item 속성이 필요합니다.

각 항목에는 다음이 포함되어야 합니다.

  • data-accordion-button과 아코디언의 ID를 가진 버튼.
  • data-accordion-title과 아코디언의 ID를 가진 선택적 제목.
  • data-accordion-body와 아코디언의 ID를 가진 본문.
 Options

Tab 클래스 생성자에 전달할 수 있는 옵션에 대한 자세한 설명입니다.

옵션명 타입 기본값 설명
id String 필수 아코디언 인스턴스의 고유 식별자입니다. 이는 HTML의 주 아코디언 컨테이너 요소에 있는 data-accordion 속성과 일치해야 합니다.
expanded string 또는 null null 컴포넌트 로드 시 어떤 아코디언 항목이 초기에 확장되어야 하는지 지정합니다. 확장하려는 아코디언 버튼의 ID를 제공해야 합니다. null로 설정하면 기본적으로 어떤 항목도 확장되지 않습니다.
singleOpen boolean true 한 번에 하나의 아코디언 항목만 열릴 수 있는지(true) 또는 여러 항목이 동시에 열릴 수 있는지(false)를 결정합니다. true인 경우, 한 항목을 열면 현재 열려 있는 다른 항목이 자동으로 닫힙니다.
